# Portfolio — Md Karimul Islam

A modern, dark, minimal personal site for an AI/ML engineer. Built with **Next.js 16**,
**React 19**, and **Tailwind CSS v4**. Features an **"Ask my AI"** chat widget (a Groq-backed
assistant that answers questions about me) and one-tap **WhatsApp** contact.

## Highlights

- **Ask my AI** — a chat bubble where recruiters can ask about my projects, research, and
  availability. Answers are generated by an LLM grounded in my profile, via a secure
  server-side Route Handler (`app/api/chat/route.ts`) — the API key never reaches the browser.
- **Live project showcase** — my four deployed apps (text, vision, audio, MLOps) with demo + code links.
- **Publications, skills, experience** — all content in one editable file (`lib/data.ts`).
- **WhatsApp click-to-chat** and email/LinkedIn contact.

## Run locally

```bash
npm install
cp .env.example .env.local     # add your free Groq key from https://console.groq.com
npm run dev                    # http://localhost:3000
```

## Configuration

| Variable | Purpose |
|---|---|
| `GROQ_API_KEY` | Powers the "Ask my AI" widget (server-side only) |

## Deploy (Vercel)

Push to GitHub, import the repo on [Vercel](https://vercel.com), add `GROQ_API_KEY` as an
environment variable, and deploy. Every push auto-deploys.

## Editing content

Everything (projects, publications, skills, experience, links) lives in
[`lib/data.ts`](lib/data.ts). The AI assistant's knowledge is in
[`lib/profileContext.ts`](lib/profileContext.ts). To add a photo, replace the monogram block
in [`components/Hero.tsx`](components/Hero.tsx) with an `<img>`.
